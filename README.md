# ByTeora Pos Backend App

ByTeora POS Backend App adalah aplikasi backend untuk sistem Point of Sales (POS) yang dibangun menggunakan Golang dan Gin Framework. Proyek ini dirancang untuk menyediakan layanan API yang cepat, aman, dan scalable, mendukung operasi utama pada sistem POS seperti manajemen produk, transaksi, pengguna, dan laporan penjualan.
